# mr-tools
This is Simple Bash Script for Installing Tools from GO.



# How to install ?
```
git clone https://github.com/Imran407704/mr-tools.git
```

```
cd mr-tools
```
```
chmod +x mr-tools.sh
```

All in 1 Command


```
git clone https://github.com/Imran407704/mr-tools.git; cd mr-tools; chmod +x mr-tools.sh
```

How to run ?

```
./mr-tools.sh
```
